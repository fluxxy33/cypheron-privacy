1. Introduction

This Privacy Policy explains how Cypheron ("the bot") collects, uses, and stores data from users on Discord servers where the bot is invited.

By using the bot, you agree to this policy.

2. Data Collected

  • User IDs and Server IDs

  • Voice or text activity (for XP or ranking systems)
  • Server configuration settings

3. Use of Data

Collected data is used solely for:

  • Providing and improving bot features (e.g., ticket system, XP, shop)

The data is not sold, shared, or used for advertising purposes.

4. Data Storage

Data is securely stored using JSON files on a private server. Access is restricted to the developer only.

5. Data Retention and Deletion

  • Data is retained only as long as necessary for bot operation.
  • Server admins may request data deletion at any time by contacting the developer.
  • If the bot is removed from a server, its data can be deleted upon request.

6. Third Parties

This bot does not share data with third parties. It uses the Discord API and follows Discord’s Privacy Policy.

7. Your Rights
